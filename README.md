### Back-End API

Used Node.js, Express, PostgreSQL, Sequelize, Elastic Search, Winston Logger

### Run locally

Follow the steps in root file Back-End API to run the Back-End first

```
npm install
nodemon dev
```

Then run Front-End in client file
```
npm install
npm run serve
```
